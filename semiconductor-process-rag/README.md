# Semiconductor Process Troubleshooting RAG

本專案為「領域驅動 RAG 系統規劃書」期中作業，主題為半導體製程異常分析與知識檢索輔助。

## Project Goal
本系統旨在協助使用者針對半導體製程中的異常現象、缺陷模式、設備警報、製程參數偏移與修正建議進行可解釋、可追溯的檢索增強生成（RAG）。

## Core Design Principles
- Domain-driven knowledge representation
- Hybrid retrieval over vector + keyword + metadata
- Parent-document retrieval with reranking
- Grounded generation with citation and uncertainty disclosure
- Evaluation-first development
